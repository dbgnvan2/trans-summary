# Key Terms Extraction

You are an expert academic analyst. Your task is to extract key domain-specific terminology from the provided transcript.

## Instructions

1.  **Identify Key Terms:** Find 5-10 significant, domain-specific terms or concepts discussed in the lecture.
2.  **Define:** For each term, provide a clear, concise definition based on how it is used in the transcript.
3.  **Format:**
    - Create a sub-header for each term: `### Term Name`
    - Write the definition below the header.

## Output Format

### Term 1
Definition of term 1.

### Term 2
